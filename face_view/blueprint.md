# FaceView: 관상으로 보는 나의 운명 (Program Blueprint)

## 1. 프로젝트 개요
**FaceView**는 현대적인 웹 기술과 인공지능 얼굴 인식 라이브러리를 결합하여, 사용자의 얼굴 특징을 분석하고 이를 전통적인 관상학적 관점에서 해석해 주는 디지털 관상 분석 플랫폼입니다. 카메라를 통한 실시간 촬영이나 사진 업로드를 통해 누구나 쉽고 재미있게 자신의 운세를 확인할 수 있습니다.

## 2. 주요 개념
이 프로그램은 사용자의 얼굴에서 추출된 68개의 주요 랜드마크(눈, 코, 입, 턱선 등)를 바탕으로 관상학적 특징을 시뮬레이션합니다. 
- **관상학적 해석**: 얼굴의 형태와 이목구비의 비율을 분석하여 성격과 미래의 운을 예측합니다.
- **사용자 경험**: 복잡한 절차 없이 촬영 한 번으로 7가지 핵심 운세 리포트를 즉시 제공합니다.

## 3. 핵심 기능
1. **실시간 카메라 캡처**: 브라우저의 MediaDevices API를 사용하여 사용자의 얼굴을 실시간으로 스트리밍하고 분석용 이미지를 캡처합니다.
2. **사진 업로드 지원**: 기존에 촬영된 사진 파일을 업로드하여 분석할 수 있는 기능을 제공합니다.
3. **AI 얼굴 분석**: `face-api.js`를 활용하여 얼굴의 위치와 이목구비의 좌표를 정밀하게 찾아냅니다.
4. **7가지 카테고리 운세 리포트**:
    - **성격**: 얼굴형과 전반적인 인상을 통한 성향 분석
    - **운명**: 삶의 전반적인 흐름과 개척 정신
    - **재물운**: 코의 모양과 균형을 통한 경제적 운세
    - **연애운**: 눈매와 눈빛을 통한 인연운
    - **자식운**: 입술과 하관의 형태를 통한 가족운
    - **직장운**: 턱선과 얼굴의 골격을 통한 사회적 성취도
    - **미래**: 노후의 평온함과 최종적인 삶의 결실
5. **반응형 UI/UX**: 모바일과 데스크톱 환경 모두에서 최적화된 화면을 제공하며, 세련된 애니메이션 효과로 분석 과정의 몰입감을 높입니다.

## 4. 기술 스택
- **Frontend**: HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+)
- **AI Library**: `face-api.js` (얼굴 인식 및 랜드마크 감지)
- **Icons**: Font Awesome 6.0
- **Deployment**: 단일 페이지 애플리케이션(SPA) 구조로 어디서든 간편하게 실행 가능

## 5. 기대 효과 및 활용
- **엔터테인먼트**: 친구, 가족과 함께 관상을 보며 즐거운 대화 소재로 활용
- **자기 탐색**: 자신의 인상에 대한 객관적인(?) 피드백과 재미있는 운세 확인
- **기술 실습**: 웹 환경에서의 AI 라이브러리 활용 및 이미지 처리 기술 구현 사례
